## En une phrase

Le produit est **excellent en démonstration** et **pas encore prêt pour les
données réelles** (il manque un backend). Ces documents disent exactement où on
en est, sans enjoliver — chaque affirmation est soit une commande exécutable,
soit une limite assumée et datée.
